# Sortieralgorithmen

Es sollen verschiedene Sortieralgorithmen implementiert und deren Laufzeiten verglichen werden.

## Insertion Sort

Programmiere den Insertion Sort Algorithmus. Dabei soll eine unsortierte Liste von Zahlen als Eingabe genommen und eine sortierte Liste als Ausgabe zurückgegeben werden. Die Eingabeliste soll unverändert bleiben. Programmiere Tests mit pytest, um die Korrektheit der Implementierung zu überprüfen.

## Selection Sort

Programmiere den Selection Sort Algorithmus. Dabei soll eine unsortierte Liste von Zahlen als Eingabe genommen und eine sortierte Liste als Ausgabe zurückgegeben werden. Die Eingabeliste soll unverändert bleiben. Programmiere Tests mit pytest, um die Korrektheit der Implementierung zu überprüfen.

## Bubble Sort

Programmiere den Bubble Sort Algorithmus. Dabei soll eine unsortierte Liste von Zahlen als Eingabe genommen und eine sortierte Liste als Ausgabe zurückgegeben werden. Die Eingabeliste soll unverändert bleiben. Programmiere Tests mit pytest, um die Korrektheit der Implementierung zu überprüfen.

## Merge Sort

Vergleiche die Laufzeiten der implementierten Sortieralgorithmen anhand von Listen unterschiedlicher Längen und Inhalten (z.B. zufällige Zahlen, bereits sortierte Listen, umgekehrt sortierte Listen). Dokumentiere die Ergebnisse in einer Tabelle oder einem Diagramm.
