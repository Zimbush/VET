# Projektkontext

Es handelt sich um ein Entwicklungsprojekt mit Python.

## Projektinfrastruktur

- Das Projekt soll mit uv verwaltet werden.
- uv soll eine virtuelle Umgebung erstellen.
- Es soll Python Version 3.14 eingesetzt werden.
- Es soll einen src-Ordner für den funktionalen Quelltext geben.
- Es soll einen test-Ordner für Unittests geben.
- Es soll mit pytest getestet weren.
